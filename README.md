# YT Recs Word Cloud

Word clouds generated from my YouTube homepage recommendations, colored by [NRC EmoLex](http://saifmohammad.com/WebPages/NRC-Emotion-Lexicon.htm) emotion data (and earlier runs, Plutchik).

Each run captures the video titles shown on my YouTube homepage and renders them as a self-contained HTML word cloud, colored by dominant emotion.

- [`index.html`](index.html) — archive of all runs, newest first
- [`YouTubeWordClouds/`](YouTubeWordClouds) — individual word cloud pages, one per run

New runs are added regularly.
